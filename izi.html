Izi photo mosaic
Every day milion of people surf on the net. The goal of this web experience is to make more people collaborate in a productive way.
Why should people collaborate on the net?
In the world there are a lot of fanbases that share their passion. Whenever a singer is about to release a music album, fans go to great lengths to find out something about it. Here comes the idea to create a “photo mosaic”: an album cover created by many small images. Fans have to send a picture of themselves to discover the album cover, because this picture will be one of the small images that will complete the album cover.
Team:
- 4 students

My main role:
- UI/UX designer
- Programmer
- Visual designer

Tools
- HTML, CSS
- P5 library
- Firebase
The process
1. Define the user journey and tools
The goal was to create a simple and intuitive web experience, so we designed only 3 pages, one with instructions, one where people can take the picture and one with the mosaic. We also defined the software scheme: everytime that a person send a picture, it will be stored in a digital space (Firebase) and than uploaded in the mosaic in the right position.

2. Define the graphics
We decided to use a simple graphic that remebers the singer’s style. We wanted to enhance the mosaic.




3. Create the algorithm
We use the P5 library to code this project. It wasn’t easy to create the algorithm hat would allow the creation of the mosaic, infact we did a lot of tests. At the end we created an algorithm that inserts the user picture in the position where there is the highest brightness affinity.

4. Try it!
We publish the website but the majority of picture were made by ourselves. Only a small group of people tried it, everyone was impressed by the result: the mosaic was very similar to the album cover. Try it yourself!
